## 📌 About the Project

This is a modern, responsive portfolio website built to highlight:

* My projects 💻
* My technical skills ⚡
* My experience 📊
* Contact information 📬

The website includes smooth animations, interactive UI, and a clean design to create a great user experience.

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 CSS3
* 🎬 GSAP (Animations)
* 📜 JavaScript (ES6+)

## ✨ Features

* 🎯 Smooth scrolling navigation
* 🎬 GSAP animations
* 📱 Fully responsive design
* 🧩 Interactive project cards
